# Jinyeop Song - Personal Website

Personal academic website built with [Hydejack](https://hydejack.com/) theme.

## Local Development

```bash
brew install ruby@3.2
export PATH="/opt/homebrew/opt/ruby@3.2/bin:$PATH"
gem install bundler:2.6.3
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000

If you use `rbenv` or `asdf`, this repo pins Ruby `3.2.8` in `.ruby-version`.
